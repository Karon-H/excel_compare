# Excel对比工具 (Excel Compare Tool)

一个基于 Python, PyQt5 和 Pandas 开发的高性能、可视化 Excel 文件比对工具。支持大数据量处理、多种比对模式以及直观的差异高亮显示。

## 核心特性

- **双模比对引擎**：
    - **主键模式**：基于用户指定的关键列进行行对齐，完美处理行乱序、新增或删除的情况。采用 Pandas 矢量化引擎，支持数十万行数据的秒级比对。
    - **序列模式**：基于 `difflib` 算法，适用于无固定主键、结构相似的文档比对。
- **现代化可视化界面**：
    - **双窗同步滚动**：左右并排展示新旧文件，支持同步缩放、选中与滚动。支持 **Shift + 鼠标滚轮** 快速横向滚动。
    - **智能表头识别**：默认自动寻找第一个非空行作为表头，支持手动指定表头行号，并新增 **“无表头”模式**（自动生成列名），全方位适配各类 Excel 结构。
    - **差异清单模式**：独立页签以清单形式列出每一项具体变更（主键、列名、旧值、新值），告别宽表格横向滚动的烦恼。
    - **统计仪表盘**：比对完成后自动弹出可视化统计面板，通过饼图分析新增、删除、修改的分布情况。
- **深度视图控制**：
    - **多行多列冻结**：支持自定义冻结行数和列数，方便在大表格中固定表头或关键列。
    - **实时搜索与过滤**：支持对比对结果进行全文搜索，或按状态（一致、修改、新增、删除）快速筛选。
- **极致的稳定性与兼容性**：
    - **中文路径支持**：针对 Windows 环境优化，支持在包含中文的路径下运行与打包。
    - **异构表格处理**：支持列结构不一致的文件比对，缺失列自动补全。
    - **智能 Sheet 识别**：支持模糊匹配 Sheet 名称，自动处理表头偏移与重复列名。

## 技术栈

- **语言**：Python 3.8+
- **GUI 框架**：PyQt5 (Model-View 架构)
- **数据处理**：Pandas, NumPy
- **Excel 引擎**：Openpyxl, Xlrd
- **打包系统**：自定义 PyInstaller 构建脚本

## 安装说明

1.  **克隆仓库**：
    ```bash
    git clone [repository-url]
    cd excel-compare
    ```

2.  **安装依赖**：
    ```bash
    pip install -r requirements.txt
    ```

3.  **运行程序**：
    ```bash
    python main.py
    ```

## 使用指南

1.  **选择文件**：点击“选择旧文件”和“选择新文件”按钮加载需要比对的 Excel。
2.  **配置参数**：
    - 选择需要比对的子表（Sheet）。
    - 选择比对模式：若数据有唯一标识（如 ID），建议使用“主键模式”并勾选关键列。
3.  **执行比对**：点击“开始比对”，等待进度条完成。
4.  **查看结果**：
    - 在“主视图”中通过颜色查看差异。
    - 在“差异项总览”中快速查看所有变更点。
    - 在右侧“详情面板”查看选中行的具体差异明细。
5.  **视图调整**：通过顶部面板调整行高、列宽，或设置冻结行列。
