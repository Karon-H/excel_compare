版本: V1.0.9
日期: 2025-12-26
更新内容:
1. 项目初始化，完成 Excel 基础比对功能。
2. 支持多 Sheet 选择。
3. 支持合并单元格自动填充。
4. 支持行高和列宽动态调整。
5. 按照参考规则重构项目结构。
6. 完成标准化打包工程构建系统，支持自动提取版本号、Spec 打补丁及产物重命名。
7. 根据《ico图标使用规则》完善程序图标设置方案。
8. 优化差异项标记：
    - 将差异单元格的标记符号由 `【】` 改为更醒目的 `>>> <<<`。
    - 增强了 Treeview 中差异项的识别度。
9. 响应“高亮仅对单元格标记”需求：
    - UI 彻底重构：弃用原生 Treeview 控件，改用自定义 `ScrollableTable` 组件。
    - **UI 单元格高亮**：实现了 UI 界面中发生变化的特定单元格浅红色填充，并配合深红色文字增强显示。
    - 导出增强：导出 Excel 依然保持高标准单元格级高亮。
    - 修复了 V1.0.4 中 UI 无法显示单元格背景色的问题。
10. 滚动交互优化：
    - 优化左右水平滚动条布局，使用网格布局确保对齐。
    - **鼠标滚轮支持**：支持鼠标滚轮垂直滚动，支持 `Shift + 鼠标滚轮` 进行水平滚动。
    - **同步滚动增强**：新增“垂直滚动同步”勾选框，可自由切换是否同步垂直滚动。
    - 优化了同步滚动的底层逻辑，防止递归调用。
11. 用户体验与功能增强：
    - **加载动画弹窗化**：将比对渲染过程中的加载动画由内嵌状态栏改为**模态弹窗**形式，视觉反馈更直接。
    - **异步比对**：将比对任务移至后台线程执行，确保 UI 界面在比对过程中不会出现“无响应”状态。
    - **菜单栏**：新增菜单栏，包含“使用说明”和“版本说明”，方便用户快速上手。
    - **默认设置优化**：将左右滚动同步默认设为开启。
12. 渲染性能与视觉优化：
    - **虚拟化滚动渲染**：彻底重构了 `ScrollableTable` 组件，采用“虚拟化”技术，仅对可见区域进行控件渲染。极大提升了大文件（上千行数据）下的滚动流畅度，解决了“拖动快了来不及渲染”的问题。
    - **状态列色彩强化**：对第一列“状态”进行了背景颜色填充（新增-绿、删除-黄、修改-蓝），增强了行级差异的辨识度。
    - **智能列宽**：新增列宽预计算逻辑，根据数据内容自动调整列展示宽度。
    - **图例更新**：同步更新了 UI 下方的图例说明，清晰区分“行状态”与“单元格差异”。
